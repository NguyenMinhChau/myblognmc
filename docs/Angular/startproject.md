## Cài đặt Angular CLI

```bash
npm i -g @angular/cli
```

> **Check version angular CLI:** `ng version`

## Tạo dự án

```bash
ng new may-app
```

> **/src/app/app.components.html:** file này viết như html nhưng không bao gồm cấu trúc cơ bản của html.

## Chạy dự án

```bash
cd may-app
ng serve
```
